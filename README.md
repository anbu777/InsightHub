Setup)Ikuti langkah ini secara berurutan untuk menjalankan proyek ini di komputer lokal Anda.1. Persiapan LingkunganClone repository:git clone [https://github.com/zaidanjibran/insight-hub.git](https://github.com/zaidanjibran/insight-hub.git)
cd insight-hub
Install dependencies:npm install
Siapkan Docker (Wajib):Install dan jalankan Docker Desktop.PENTING: Pastikan status Docker adalah "Running" (Hijau). Docker diperlukan untuk men-deploy Supabase Edge Functions dan menjalankan database lokal (jika menggunakan local supabase).Install Supabase CLI:npm install supabase --save-dev
2. Konfigurasi EnvironmentBuat file .env.local di root proyek dengan menyalin dari .env.example. Minta kunci API (API Keys) yang diperlukan kepada pemilik proyek sebelumnya.cp .env.example .env.local
Pastikan variabel berikut terisi:NEXT_PUBLIC_SUPABASE_URL & ANON_KEYSUPABASE_SERVICE_ROLE_KEY (Penting untuk fungsi admin & Edge Functions)RESEND_API_KEY (Untuk notifikasi email)NEXT_PUBLIC_TURNSTILE_SITE_KEY & SECRET_KEY (Opsional, untuk keamanan form)3. Setup Database & BackendAnda memiliki dua pilihan: Menghubungkan ke Proyek Cloud yang sudah ada, atau menjalankan secara lokal. Panduan ini untuk koneksi ke Proyek Cloud.Login ke Supabase CLI:npx supabase login
Hubungkan ke Proyek Cloud:npx supabase link --project-ref <PROJECT-ID-ANDA>
(Dapatkan Project ID dari Dashboard Supabase > Settings > General).Terapkan Struktur Database (Schema):Jalankan perintah ini untuk membuat semua tabel, fungsi, dan kebijakan keamanan berdasarkan file schema.sql yang sudah disediakan:# Salin isi file supabase/schema.sql dan jalankan di SQL Editor Dashboard Supabase
# Atau gunakan CLI jika sudah setup migrasi
(Note: Cara termudah adalah copy-paste isi supabase/schema.sql ke SQL Editor di dashboard).Isi Data Awal (Seed):Agar aplikasi tidak kosong, isi dengan data dummy.(Copy-paste isi supabase/seed.sql ke SQL Editor dan jalankan).Deploy Edge Function (PENTING):Jalankan perintah ini agar fitur Upload CSV Otomatis bisa berjalan. Docker harus menyala.npx supabase functions deploy process-csv-upload --no-verify-jwt
